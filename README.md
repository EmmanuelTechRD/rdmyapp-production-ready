# Django App Deployed on Heroku and AWS

Django app with server and PostgreSQL database deployed on Heroku and image serving with AWS S3 buckets. This app was created to demonstrate the simplicity of using the Python Django (Web Development Framework) on a small project, showcasing its ability to deliver good performance when deployed on web services platforms.
